## 📖 Overview

**Sociogram Test** is a web-based application developed for a professor at **Universidad Tecnológica de Puebla (UTPuebla)**.

The project was designed to analyze social relationships within student groups by applying **two sociometric questionnaires**, each containing **10 questions**. Based on the responses, the system automatically calculates scores and identifies the students who receive the highest number of selections in each category.

The application was used to evaluate **four different groups** from the Information Technologies department.

---

## ✨ Features

- 📝 Two sociometric questionnaires
- 📊 Automatic score calculation
- 👥 Group-based evaluation
- 🏆 Identifies the most selected students
- 💾 Stores all responses in a MySQL database
- 📈 Simple and intuitive interface for teachers

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/sociogram-test.git
```

2. Import the MySQL database.

3. Configure the database connection in:

```php
application/config/database.php
```

4. Configure the project URL in:

```php
application/config/config.php
```

5. Start your local server (XAMPP, MAMP, Laragon, etc.).
